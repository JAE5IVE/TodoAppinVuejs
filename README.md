# TodoFlow Vue

A Vue 3 conversion of the original TodoFlow TypeScript/React project.

## Live Demo

https://todoappinvuejsts.netlify.app/

## Features

- LocalStorage-backed todo persistence
- Category filters for work, personal, urgent, shopping, and health tasks
- Search across task titles
- Add, complete, and delete tasks
- Edit existing tasks
- Created dates and optional due dates
- Offline/online network indicator
- Toast notifications
- Export tasks to JSON and clear all saved tasks
- Dark mode with saved theme preference
- Responsive Vue single-file component UI

## Tech Stack

- Vue 3
- Vite
- lucide-vue-next
- Plain CSS
- Vitest and Vue Test Utils

## Getting Started

```bash
npm install
npm run dev
```

Build for production:

```bash
npm run build
```

Run tests:

```bash
npm test
```
