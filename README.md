# TodoFlow Vue

A Vue 3 conversion of the original TodoFlow TypeScript/React project.

## Live Demo

https://todoappinvuejsts.netlify.app/

## Features

- LocalStorage-backed todo persistence
- Category filters for work, personal, urgent, shopping, and health tasks
- Search across task titles
- Add, complete, and delete tasks
- Offline/online network indicator
- Toast notifications
- Responsive Vue single-file component UI

## Tech Stack

- Vue 3
- Vite
- lucide-vue-next
- Plain CSS

## Getting Started

```bash
npm install
npm run dev
```

Build for production:

```bash
npm run build
```
